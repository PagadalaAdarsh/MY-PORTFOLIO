Did Any Main Project?


Image Enhancement with Application of Local and Global Enhancement Methods for dark Images.
Project Goal : In my final year project, I focused on improving the quality of underexposed images. The project aimed to enhance dark images by applying two types of techniques that are local and global enhancement methods.

Tools and Techniques Used : For Global Enhancement, I used Histogram Equalization, Contrast Stretching, Gamma Correction which improves the overall brightness of the image. For Local Enhancement, I used Adaptive Histogram Equalization, Local Contrast Enhancement, which works on specific areas of the image to improve details darker regionswithout effecting the bright areas.
Outcome : At the end the project, the enhanced images were much clearer, with improved visibility in dark areas. This project helped me to learn how to combile different image processing techniques to solve real-world problems which is useful in fields like photography, medical imaging, and security.
